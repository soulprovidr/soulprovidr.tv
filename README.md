# SOULPROVIDR.TV

## Quick start
1. Rename `default.env` to `.env` and set variables.

1. Install `save-video` dependencies:
`yarn lambda:install`

1. Deploy `save-video` to Lambda:
`yarn lambda:deploy`

1. Build client:
`yarn app:build`

1. Deploy client:
`yarn app:deploy`

1. Sync playlists:
`yarn sync-playlists`
